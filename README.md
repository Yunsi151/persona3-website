# persona3-website

A personal portfolio website styled as a Persona 3 (JRPG) UI, built with React 19 and Vite.

## Assets

The project requires media assets (videos and images) that are not included in this repository. Download them and place them in `src/assets/` before running the project:

**[Download assets from Google Drive](https://drive.google.com/drive/folders/1_dVxHVTJO_YtMAo9ia7wnja1XY0nImr5?usp=drive_link)**

## Getting Started

```bash
npm install
npm run dev
```

## Scripts

| Command | Description |
|---|---|
| `npm run dev` | Start dev server |
| `npm run build` | Production build |
| `npm run build:watch` | Watch mode build |
| `npm run lint` | Run ESLint |
| `npm run preview` | Preview production build |

## Stack

- React 19, React Router v7
- Framer Motion (page transitions)
- Vite 8
